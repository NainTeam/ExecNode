# ExecApi
Api to execute commands in server

####Languages:
- Python (HttpServer)
- Node (Express)

###What it is?
It's a simple api that handle a command and execute it in shell. Some kind of backdoor that can be used with simple post request.

- Default port: 666
- Default route: /exec
